# Data Model for Movie Collection App

## Movie

- **title**: string
- **genre**: string
- **director**: string
- **cast**: array of strings
- **year**: integer
- **format**: string (e.g., "Blu-ray", "DVD", "4K UHD")
- **imdb_rating**: float
- **rotten_tomato_rating**: integer
- **plot**: string
- **acquired_date**: date
